# Prompt

좋아. Github Pages + 상태 내보내기/가져오기 방식으로 결정할게. 다만, 그 전에 진행해야 하는 것들이 있어.

## 요청

- 다음과 같이 작업을 진행하기 위한 계획을 세워줘.
- 작성된 계획을 포함한 문서를 루트 폴더에 "Publish_Plan.md" 파일로 생성해줘.

## 작업순서

- 저장소 정리 : 저장소를 전체 점검해서, 폴더 및 파일 구조를 체계화해줘.
- README.md 작성 : Github에 퍼블리시하기 전, 이 저장소의 컨셉과 애플리케이션의 용도, 기술사양 등을 서술한 README.md 문서를 작성해줘.
- Github 퍼블리시 : 이 저장소를 원격에서도 작업할 수 있도록 Github에 퍼블리시해줘.
- Github Pages + 상태 내보내기/가져오기 개발
